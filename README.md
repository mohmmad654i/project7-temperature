# project7-temperature 
# 🌡️ نظام مراقبة درجات الحرارة | Temperature Monitor System

مرحباً! هذا هو المشروع السابع (**Project 7**) في رحلتي لتعلم البرمجة وهندسة الأنظمة.

## 📖 نظرة عامة (Overview)
هذا البرنامج مكتوب بلغة **C++**، وهو مصمم لاستقبال درجات الحرارة لمدة أسبوع (7 أيام) وتحليلها برمجياً لمساعدة المستخدم في مراقبة الحالة الجوية وإصدار تنبيهات ذكية.

## 🚀 المميزات (Features)
* **Data Analysis:** حساب أعلى درجة حرارة (**Maximum**) وأدنى درجة (**Minimum**).
* **Average Calculation:** حساب متوسط درجات الحرارة خلال الأسبوع.
* **Smart Alert:** إصدار تنبيه (**Beware**) في حال تجاوزت الحرارة حد الـ 40 درجة مئوية.
* **Simple Interface:** واجهة تفاعلية سهلة الاستخدام عبر الـ Console.

## 🛠️ التقنيات (Tech Stack)
* **Language:** C++
* **Logic:** المصفوفات (Arrays)، الحلقات التكرارية (Loops)، والجمل الشرطية (Conditions).

## 💻 طريقة التشغيل (How to Run)
1. قم بتشغيل الكود باستخدام أي مترجم ++C (مثل Dev-C++ أو VS Code).
2. أدخل درجات الحرارة المطلوبة لكل يوم.
3. استلم التقرير النهائي والتنبيهات فوراً.
